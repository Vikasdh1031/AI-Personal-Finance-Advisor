# AI Personal Finance Advisor 💰🤖

## Overview

AI Personal Finance Advisor is a full-stack financial management application that helps users track expenses, manage spending habits, and receive AI-powered financial guidance.

The system combines Spring Boot, MySQL, and Google's Gemini AI to provide intelligent financial recommendations through an interactive chat interface.

---

## Features

* Add and manage expenses
* Categorize spending records
* Store financial data in MySQL
* AI-powered financial advice using Gemini API
* Interactive chat-based assistant
* Expense tracking and analysis
* REST API architecture
* Responsive web interface
